# Polytoria-Assets
This repository includes free assets and models for the communities of Polytoria.

## Addons
After you've installed an addon, import them into your project. Then, select and right click them and choose the "Install as addon" option. 
Now, you can use it to your will. The debug prints from the addon are contained inside the creator console (not to be confused with the client console!).
If you've seen errors, please report it as an issue to this repository as soon as possible!
